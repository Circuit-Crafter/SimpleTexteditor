# SimpleTexteditor

A simple text editor written in Python for editing plain text files like .txt and .md.

## Installation

```bash
sudo apt install ./simpletexteditor
```

## Usage

```bash
simpletexteditor
```

Or open it from your applications menu.

## Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| New File | Ctrl+N |
| Open File | Ctrl+O |
| Save | Ctrl+S |
| Quit | Ctrl+Q |
| Select All | Ctrl+A |
| Undo | Ctrl+Z |
| Redo | Ctrl+Y |
